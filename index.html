Projects

Below you will find links to some of the past projects I have worked on

Live Music Hub

This is the first group project I worked on. It was created using Vanilla JavaScript, HTML, CSS and Bootstrap. I was able find to local concert venues and upcoming concerts by using Google Maps, Google Search APIs and the API from Jambase. The user enters a specific zip code and search radius into the search bar and the venue or concert is displayed on the right. The user clicks on the venue or concert they would like information on, then it would pop on the map with a red flag on its location.

LiveActive

This was the second group project I helped work on. This project was built with idea to help the user live a healthy active lifestyle. The user needs to create an account which will allow the user access to local meetups and healthy eating options by using the APIs from Meetup and food2fork. The user can locate local outdoor activities by conducting a search within a specific area and the type of activity they would like to participate in. The user can also search for healthy eating options of the foods they enjoy to eat.


Final Countdown

This project was built with React.js and utilized APIs from NASA, SpaceX, Virgin Gallactic and Space News. The concept behind this project was to give the average person access to enjoy traveling to space. Once the user has logged in they are able to view space travel options, frequently asked questions, reviews and the photo gallery. When a user has decided which of the travel options best matches their needs, they will be directed to a payment page. Using Stripe, the user can purchase the plan and a confirmation email will be sent to the user.
